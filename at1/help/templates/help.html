Help

Have a problem?

Here are some issues you may have while trying to use Flasher.

I can't login

Either your password or username is wrong, or that user doesn't exist. You can sign up with another account.

I can't sign up

The username you are using already exists. Try signing up with another username.

I keep getting redirected to Change Deck Directory when I go to Roulette

The deck you are using must be damaged. Type in the deck directory again or select a different deck.
